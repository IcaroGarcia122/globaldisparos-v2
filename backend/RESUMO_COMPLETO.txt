═══════════════════════════════════════════════════════════════════
🛡️ SISTEMA ANTI-BAN - FUNCIONALIDADES
═══════════════════════════════════════════════════════════════════

1. ✅ LIMITES INTELIGENTES POR IDADE DA CONTA
   • Conta Nova (< 7 dias): 50 msgs/dia, delay 15-45s
   • Conta Média (7-30 dias): 150 msgs/dia, delay 8-20s
   • Conta Antiga (> 30 dias): 500 msgs/dia, delay 3-10s

2. ✅ VARIAÇÃO AUTOMÁTICA DE MENSAGENS (4 variações)
   • Original: "Olá! Tenho uma proposta."
   • + Emoji: "Olá! Tenho uma proposta. 😊"
   • + Prefixo: "Oi! Olá! Tenho uma proposta."
   • + Despedida: "Olá! Tenho uma proposta. Abraço!"

3. ✅ VARIÁVEIS PERSONALIZADAS
   • {{nome}}, {{telefone}}, {{empresa}}, {{cidade}}
   • {{data}}, {{hora}}, {{dia_semana}}, {{mes}}, {{ano}}

4. ✅ DELAYS RANDÔMICOS COM VARIAÇÃO
   • Base aleatória + variação de ±20%
   • NUNCA repete o mesmo delay

5. ✅ PAUSAS INTELIGENTES (Burst Control)
   • A cada 5-20 mensagens → pausa de 2-5 minutos
   • Simula comportamento humano

6. ✅ HORÁRIO COMERCIAL (9h-21h)
   • Só envia durante horário comercial
   • Pausa automática à noite
   • Retoma sozinho às 9h

7. ✅ DETECÇÃO AUTOMÁTICA DE BAN
   • Monitora taxa de erro (>70% = ban)
   • Pausa campanha automaticamente
   • Notifica usuário

8. ✅ DASHBOARD DE ESTATÍSTICAS
   • Stats por instância
   • Stats por campanha
   • Gráficos de envio
   • Taxa de sucesso

📝 VARIÁVEIS DE AMBIENTE IMPORTANTES
═══════════════════════════════════════════════════════════════════

Essenciais:
• DB_PASSWORD             - Senha PostgreSQL
• JWT_SECRET              - Chave secreta JWT
• ADMIN_EMAIL             - Email admin padrão
• ADMIN_PASSWORD          - Senha admin padrão
• FRONTEND_URL            - URL do frontend

Opcionais (já tem padrão):
• PORT                    - Porta servidor (3001)
• DB_HOST                 - Host DB (localhost)
• DB_NAME                 - Nome DB (whatsapp_saas)
• ANTI_BAN_*              - Configs anti-ban

═══════════════════════════════════════════════════════════════════
🚀 COMO RODAR (RESUMO)
═══════════════════════════════════════════════════════════════════

1. Instale PostgreSQL
2. Crie database `whatsapp_saas`
3. Copie .env.example → .env
4. Configure DB_PASSWORD no .env
5. npm install
6. npm run dev

Pronto! Backend em http://localhost:3001
